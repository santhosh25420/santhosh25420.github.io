# Portfolio Template

This is a simple portfolio template for GitHub Pages.

## How to Use

1. Fork or clone this repository.
2. Customize the content in `index.html`:
   - Replace "My Name" with your name.
   - Update the about section with your description.
   - Add your projects in the projects section.
   - Update contact information.
3. Customize `styles.css` for your preferred styling.
4. Commit and push your changes.
5. Enable GitHub Pages in your repository settings:
   - Go to Settings > Pages.
   - Select the source (usually main branch).
   - Your portfolio will be available at `https://yourusername.github.io/repository-name/`.

## Features

- Responsive design
- Sections for About, Projects, and Contact
- Simple navigation
